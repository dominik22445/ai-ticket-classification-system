# ai-ticket-classification-system
System wykorzystujący sztuczną inteligencję do automatycznej analizy i klasyfikacji wiadomości klientów oraz kierowania ich do odpowiednich procesów biznesowych.  Projekt wykorzystuje model OpenAI do rozpoznawania rodzaju zgłoszenia, zapisuje wyniki w Google Sheets oraz uruchamia odpowiednie działania w zależności od kategorii wiadomości.
